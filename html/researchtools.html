Resources

A list of resources for my research.

Data management

netCDF: a set of software libraries and self- describing, machine-independent data formats that support the creation, access, and sharing of array-oriented scientific data. My notes on netCDF files are here.

NCO: The netCDF Operators (NCO) comprise a dozen standalone, command-line programs that take netCDF, HDF, and/or DAP files as input, then operate (e.g., derive new data, compute statistics, print, hyperslab, manipulate metadata) and output the results to screen or files in text, binary, or netCDF formats.

3 <-> 4: netCDF version conversion tools.

netCDF4: a Python interface to the netCDF C library.

HDF5: Hierarchical Data Format version 5.

Data

Visualization

D3: JavaScript library for manipulating documents based on data. D3 helps you bring data to life using HTML, SVG, and CSS. D3 github repo.

plotly: D3 python wrapper.
